# Prompt: Close The Loop

Use this after completing a project to promote insights from Memory to Core. This is what makes the system compound.

Copy and paste this into your chat:

---

```
It's time to close The Loop. Please review my memory files and promote strong insights to the core files.

Please do the following:

1. Read all memory files:
   - M-memory/learning-log.md
   - M-memory/feedback.md
   - M-memory/decisions.md

2. Read current core files:
   - C-core/project-brief.md
   - C-core/voice-dna.md
   - C-core/icp-profile.md

3. Identify what's ready to promote:

   **From learning-log → voice-dna:**
   - Any voice pattern that worked 3+ times? Add it as a rule.
   - Any mistake that keeps repeating? Add it as a "never do this."

   **From feedback → icp-profile:**
   - Any audience language that keeps appearing? Add it to how the ICP talks.
   - Any topic that consistently resonates? Note it in ICP interests.

   **From decisions → project-brief or voice-dna:**
   - Any decision active for 3+ months? It's probably a core principle now.
   - Any brand positioning that's proven? Strengthen it in project-brief.

4. Sort B-brain/INBOX/:
   - Move any unsorted items to the right subfolder
   - Delete anything no longer relevant
   - Summarize what you sorted

5. Update the files:
   - Add promoted insights to the right C-core file
   - Mark promoted items in M-memory as "[Promoted to C-core]"
   - Clean up any outdated or contradicting entries in M-memory

6. Show me a summary:
   - What got promoted (and to which file)
   - What stayed in Memory (not ready yet)
   - What got cleaned up
   - What was sorted from INBOX

This is The Loop. It's what makes the system get smarter over time.
```

---

## When to Use This

- After completing a major project
- At the end of each week (pair with prompt 13)
- When your output starts feeling stale
- When Memory files are getting long

---

## Why The Loop Matters

Most AI systems are static. You set them up and they stay the same forever.

The ABC-TOM Loop is different. Every project teaches you something. The Loop captures those lessons and bakes them into the system permanently.

**Week 1:** Memory is empty. Output is rough.
**Week 4:** Memory has patterns. Output improves.
**Week 8:** Strong patterns get promoted to Core. Output sounds like you.
**Week 12:** The system runs itself. You just steer.

The Loop is the difference between "using AI" and "building an AI team."
