### הכנה: התקנת קבצי הפרויקט

1. הורידו את קובץ ה-ZIP ופתחו אותו.
2. גררו את התיקייה שנפתחה לתוך ה-Obsidian Vault שלכם.
3. ב-Windows? לפעמים נוצרת תיקיית ".obsidian" מיותרת בתוך ה-Vault. התעלמו ממנה.
4. ודאו שאתם רואים את כל התיקיות (A-agents, B-brain, C-core, T-tools וכו') בסרגל הצד ב-Obsidian. הן בתוך תיקיית the-system שחילצתם מה-ZIP.

### הכנה: פתיחת Claude Code

4. פתחו שיחה חדשה ב-Claude Code.
5. **חשוב:** בחרו את תיקיית the-system כתיקיית העבודה. ככה Claude יודע איפה לקרוא, לערוך וליצור קבצים.

---

### שלב 1: בחרו את המסלול שלכם

לפני שמתחילים, בחרו את המסלול שמתאים לעבודה שלכם:

**תוכן** פוסטים לרשתות חברתיות, יצירת תוכן, מותג אישי.
**מחקר** תדריכים למנהלים, ניתוח מתחרים, מחקר שוק.
**תקשורת** מיילים מקצועיים, הודעות ללקוחות, פנייה עסקית.

**עובדים עם העסק שלכם?** פתחו את `T-tools/02-prompts/` והיכנסו לתיקיית המסלול שלכם (content, research, או communication).

מכאן, כל הפרומפטים (01 עד 04) נמצאים בתיקייה הזו. פשוט עקבו אחרי הסדר.

---

### שלב 2: מלאו את המידע העסקי

פתחו את הקובץ `B-brain/brand-discovery-worksheet.md`

מלאו את הטקסט עם המידע שלכם (העתיקו-הדביקו מהמסמך שמילאתם מראש).

**אל תשנו שמות קבצים.** הפרומפטים מקושרים אליהם.

---

### שלב 3: הכינו דוגמאות כתיבה

פתחו את תיקיית הדוגמאות שמתאימה למסלול שלכם:
- **תוכן** → `B-brain/content-samples/`
- **תקשורת** → `B-brain/communication-samples/`
- **מחקר** → `B-brain/research-samples/`

הדביקו את הטקסט שלכם (עברית או אנגלית) בכל אחד מ-3 הקבצים: `sample-01`, `sample-02`, `sample-03`.

### שלב 4: הריצו את 4 הפרומפטים

**הריצו פרומפט אחד בכל פעם. חכו שיסיים לפני שמדביקים את הבא.**

כל 4 הפרומפטים נמצאים בתיקיית המסלול שלכם. פשוט לכו לפי הסדר:

**פרומפט 1. המרת Brand Discovery**

- העתיקו את כל התוכן של `01-convert-brand-discovery.md`
- הדביקו ב-Claude Code ושלחו.
- Claude קורא את ה-Brand Discovery ומעדכן אוטומטית את קבצי ה-C-core: `project-brief`, `icp-profile`, `voice-dna`.

**פרומפט 2. לימוד סגנון הכתיבה שלי**

- העתיקו את `02-learn-my-writing-style.md` והדביקו.
- Claude קורא את 3 הדוגמאות ומעדכן את קבצי ה-Core, האייג'נטים, וה-Memory.

**פרומפט 3. סנכרון האייג'נטים**

- העתיקו את `03-sync-agents-with-context.md` והדביקו.
- Claude מוודא שכל האייג'נטים מכירים ומבינים את כל הקונטקסט שהוגדר.

**פרומפט 4. יצירת הפלט שלכם**

- העתיקו את קובץ ה-`04-...` מתיקיית המסלול שלכם והדביקו.
- מלאו את הפרטים (הנושא, הרעיון, או הנמען בהתאם למסלול).
- Claude משתמש באייג'נטים, בכישורים, ובכל ההגדרות שלכם כדי ליצור את הפלט.

מה כל מסלול מייצר:

- **תוכן** פוסט לרשתות חברתיות מוכן לפרסום (LinkedIn, Facebook, Instagram, Twitter/X)
- **מחקר** תדריך מובנה מוכן לשימוש
- **תקשורת** מייל מקצועי מוכן לשליחה
