# Prompt: Create a Social Post

Copy and paste this into your chat (fill in your idea first):

---

```
I want to create a social media post. Let's start!

---

Please follow this process:
1. Ask for the post IDEA 
   
2. Ask for the Target platform
   
3. Read the workflow in T-tools/03-workflows/social-post-workflow.md to understand the process

4. Read my brand core:
   - C-core/project-brief.md
   - C-core/voice-dna.md
   - C-core/icp-profile.md
   - B-brain/content-samples/ (if available)

5. If the Platform is LinkedIn, read the Social Post skill: T-tools/01-skills/social-post-skill/social-post-skill-Linkedin.md
   
6. Read what we've learned before: M-memory/learning-log.md

7. Now work as the Copywriter (A-agents/copywriter-agent-social.md):
   - Write a first draft
   - Create a new folder in O-output/ with the next available number
   - Save your draft as copywriter-draft.md with your notes about the choices you made

6. Then work as the Gatekeeper (A-agents/gatekeeper-agent.md):
   - Review the draft against all standards
   - Save your review as gatekeeper-review.md
   - If approved, create final-post.md ready to publish
   - If not approved, explain what needs to change

7. Update M-memory/learning-log.md with what you learned from this process

Show me the final result and tell me if it's ready to publish or needs revision.
```
